PolitiColor
PolitiColor maps Twitter sentiment data in real-time for the 2016 presidential election

What is Politicolor?

PolitiColor is an app that uses the Twiter firehose and d3.js to create chroropleth maps. Each tweet is passed into a self-designed and implemented pipeline that converts a tweet to a number representing sentiment. The pipeline aggregates and averages sentiment values for tweets by location, and displays its findings on the map above.

How PolitiColor works

The Hose

twitter api etc get data

The Engine

A romantic name for an unromantic subject, the sentiment engine uses a homebrewed(change word?) algorithm using (shit whats the thing called?) to assign values to positive and negatively connotated words.

The Server

The server takes all the individual numbers generated from the tweets and crunches them into a single number representative of the state and voting district, and then assigns a candidate based on the number.

The Map

A chroropleth map is generated based on the previously generated numbers, providing an easily understood data visualization.

The Sentiment Engine explained

The Sentiment engine works by normalizing and filtering tweets, putting them in a more usable format.

Then, the cleaned data is analyzed in chunks of words and phrases.

This parsed data is then given a number, which corresponds to a color and candidate. These colors are then displayed on the map of the United States in an easily understandable format.
