What is PolitiColor?

PolitiColor is a site(?) that uses twitter's API and D3 to create chroropleth maps that portray the genenral sentiment of twitter users in that state or election district.

By using natural language processing and (WHAT ARE WE USING), tweets are parsed in terms of their sentiment toward the candiadate or political party that they write about.

For example: if you, in Pennsylvania, write "surpised and pleased sanders is doing really well in New Hampshire #feelthebern," that gets parsed and affects Pennsylvania's display color on the map.

the sentiment engine

A romantic name for an unromantic subject, the sentiment engine uses a homebrewed(change word?) algorithm using (shit whats the thing called?) to assign values to positive and negatively connotated words.

the server

The server takes all the individual numbers generated from the tweets and crunches them into a single number representative of the state and voting district, and then assigns a candidate based on the number.

the chroropleth map

A chroropleth map is generated based on the previously generated numbers, providing an easily understood data visualization.
